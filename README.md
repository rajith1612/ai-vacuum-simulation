# 🧹 AI Vacuum Cleaner Simulation

## 📌 Overview

This project simulates an **AI-powered vacuum cleaner** that can intelligently navigate an environment by detecting dust, floor types, and obstacles using deep learning models.

---

## 🎯 Key Features

* 🧠 **Dust Detection using CNN**
* 🏠 **Floor Type Classification**
* ⚠️ **Stair & Obstacle Detection**
* 🤖 **Autonomous Navigation Logic**
* 📊 Simulation-based environment testing

---

## 🛠️ Tech Stack

* Python
* TensorFlow / Keras
* OpenCV
* NumPy

---

## 🧠 How It Works

1. Input environment is simulated as a grid/map
2. CNN models analyze:

   * Dust regions
   * Floor type
   * Obstacles / stairs
3. Decision logic moves the vacuum cleaner
4. System avoids obstacles and cleans efficiently
